# Туристическая фирма

Учебный проект по дисциплине "Интегрированные среды разработки". Десктопное приложение на C# (Windows Forms) для управления данными туристической фирмы.

## Что делает

Приложение подключается к базе данных PostgreSQL и позволяет просматривать и редактировать данные о турах, сезонах, путёвках и оплатах. Данные отображаются в таблицах, реализованы фильтрация и поиск.

Помимо таблиц, в приложении есть вкладка с диаграммами:
- Круговая диаграмма — показывает долю каждого тура от общего числа проданных путёвок (в процентах).
- Столбчатая диаграмма — показывает суммарную выручку по каждому туру.

## Стек

- C# (.NET 6, Windows Forms)
- PostgreSQL
- Npgsql (подключение к базе данных)
- ClosedXML (экспорт в Excel)
- DocumentFormat.OpenXml

## Запуск

1. Установить PostgreSQL и создать базу данных `tourfirmatest`.
2. Выполнить SQL-скрипт из файла `lastbackup.sql` для создания таблиц и заполнения данными.
3. При необходимости изменить строку подключения в `Form1.cs`:
   ```
   Host=127.0.0.1;Username=postgres;Password=1234;Database=tourfirmatest
   ```
4. Открыть решение `tourfirma.sln` в Visual Studio и запустить проект.
